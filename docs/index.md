# llm-code

A production-grade CLI coding agent for local LLMs.

## Why llm-code?

- **Works with any model** — Qwen, Llama, Mistral, GPT, Claude via OpenAI-compatible API
- **Claude Code-level experience** — 20+ tools, sub-agents, MCP ecosystem, skills
- **Your data stays local** — Run against your own vLLM/Ollama instance
- **Open source** — MIT licensed, fully extensible

## Quick Install

```bash
pip install llm-code
llm-code --model qwen3.5 --api http://localhost:8000/v1
```
